Islandora

Introduction

Islandora and Fedora 4 Integration Project!

This is where the Islandora and Fedora 4 development will happen. If you would like to get involved in the community around this project, please check out the Islandora Foundation Fedora 4 Interest Group.

Repository layout

  • camel - Islandora-Sync component. Event driven middleware based on Apache Camel that synchronizes a Fedora 4 JCR with a Drupal CMS.
  • docs - Documentation
  • drupal - Drupal modules
  • install - Vagrant development environment
